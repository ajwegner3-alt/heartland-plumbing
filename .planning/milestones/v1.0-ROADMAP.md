**Status:** SHIPPED 2026-04-06

## Overview

Build a high-performance Next.js portfolio demo for Heartland Plumbing Co. — a fictional Omaha plumber — that demonstrates NSI's capability to deliver sites that load fast, rank well, and convert visitors into calls. Seven phases take the project from scaffolded Next.js skeleton to a verified 16-page live site with Lighthouse 95+ scores confirmed on the deployed Vercel URL.

## Milestone Summary

**Key Decisions:**
- Next.js App Router with SSG — every page pre-rendered; only contact form uses a Route Handler
- next/font/google for Bitter + Nunito Sans — no render-blocking CDN link tags
- No component libraries (no shadcn/ui, no framer-motion) — client JS budget under 30KB
- Resend for contact form email — Gmail SMTP blocked on Vercel serverless
- JSON-LD via plain script dangerouslySetInnerHTML — next/Script causes RSC payload duplication
- Plumber @type for schema (more specific than generic LocalBusiness)
- CSS-only FAQ accordion via details/summary — zero JS, no hydration cost
- Copper scale shifted darker for WCAG AA compliance (base #b8733a -> #8f5a2c)
- Local hero image instead of Unsplash CDN for LCP optimization
- GTM gated behind env var to prevent failing requests
- Security headers (CSP, HSTS, X-Frame-Options) via next.config.ts

**Issues Resolved:**
- Footer Millard/Bennington slug mismatch
- WCAG AA color contrast failures on text-muted and copper colors
- LCP optimization (external image -> local, ScrollReveal SSR-visible)
- Missing /services and /service-areas index pages in sitemap

**Technical Debt Incurred:**
- SERVICE_ICONS map duplicated between service pages and area pages (should extract to shared module)
- Owner avatar is CSS circle with initial letter — should be replaced with real photo
- No automated testing (unit or E2E) — acceptable for portfolio demo
- Tailwind v4 + @tailwindcss/typography compatibility unconfirmed
