## v1 Requirements

### Performance

- [x] **PERF-01**: All pages pre-rendered at build time via SSG (zero runtime compute except contact API)
- [x] **PERF-02**: Fonts (Bitter + Nunito Sans) self-hosted via next/font — no CDN link tags
- [x] **PERF-03**: All images served as WebP via Next.js Image component with explicit dimensions
- [x] **PERF-04**: Below-fold images lazy loaded; hero image priority loaded
- [x] **PERF-05**: Sharp installed for server-side image optimization
- [x] **PERF-06**: Lighthouse Performance score 95+ on deployed Vercel URL
- [x] **PERF-07**: Lighthouse Accessibility score 95+ on deployed Vercel URL
- [x] **PERF-08**: Lighthouse Best Practices score 95+ on deployed Vercel URL
- [x] **PERF-09**: Lighthouse SEO score 95+ on deployed Vercel URL
- [x] **PERF-10**: LCP under 2.5s, CLS under 0.1, INP under 200ms

### SEO

- [x] **SEO-01**: Unique title tag per page
- [x] **SEO-02**: Unique meta description per page (150-155 chars)
- [x] **SEO-03**: One H1 per page matching primary keyword intent
- [x] **SEO-04**: Proper heading hierarchy (H1 -> H2 -> H3)
- [x] **SEO-05**: Canonical tag on every page
- [x] **SEO-06**: Open Graph tags on every page
- [x] **SEO-07**: Twitter card meta tags on every page
- [x] **SEO-08**: LocalBusiness/Plumber schema on homepage
- [x] **SEO-09**: Service schema on each service page
- [x] **SEO-10**: FAQPage schema on pages with FAQ sections
- [x] **SEO-11**: BreadcrumbList schema on all interior pages
- [x] **SEO-12**: AggregateRating schema where reviews are displayed
- [x] **SEO-13**: Auto-generated sitemap.xml with all pages
- [x] **SEO-14**: Proper robots.txt
- [x] **SEO-15**: Internal links: service pages link to related services and area pages
- [x] **SEO-16**: Internal links: area pages link to all services
- [x] **SEO-17**: No orphan pages
- [x] **SEO-18**: URL structure: lowercase, hyphen-separated

### Conversion

- [x] **CONV-02**: Sticky mobile CTA bar
- [x] **CONV-03**: CTA above the fold on every page
- [x] **CONV-04**: Contact form: name, phone, service needed, zip code
- [x] **CONV-05**: Form submission sends email notification via Resend
- [x] **CONV-06**: Trust signals adjacent to every CTA and form
